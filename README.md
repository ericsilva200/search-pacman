# search-pacman
Implementação do jogo do Pacman com algoritmos da Inteligência Artificial

Algoritmos:
  bfs = breadthFirstSearch
  dfs = depthFirstSearch
  ucs = uniformCostSearch
  gdy = greedySearch
  astar = aStarSearch
  mdp = MDPTable

Comando: 
  python3 pacman.py -l gridWorld -p SearchAgent -a fn=algoritmo
